Basalt

Basalt's primary REST API

Backend for users, projects, and bug reports.

Endpoints


GET

/users

Fetch all users

GET

/users/:id

Get user by id

POST

/users

Create new user doc - sign up

POST

/user/update/profile_pic/:id

Update an user's profile picture

POST

/user/update/bio/:id

Update an user's bio

POST

/user/update/skills/:id

Add/remove an user's display skills

DELETE

/users/:id

Delete an user document by id

GET

/projects

Fetch all projects

POST

/project/create

Create a new project document - initialize new team as an admin

DELETE

/project/:id

Delete a project by id

DELETE

/projects

Delete all projects
